# awesome-roam-themes
> a themes list of Roam Research


## how to use

1. Inside Roam Research, create a page named exactly `roam/css`
2. Create a bullet point
3. Create a code block of CSS
4. Add this code: `@import url('your-roam-research-theme.css');`
